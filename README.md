# React template

- Redux toolkit
- Redux persist
- React Router v6
- React i18n (react-i18next)
- Material UI (MUI)

## Set Up
`yarn`

Add .env file

```
REACT_APP_API_BASE_URL=http://localhost:8888/api/v1
REACT_APP_PROJECT_NAME=UI Header title
```

## Development

`yarn start`

## Build

`yarn build`
